Notion API を使って「知らないことリスト」データベースを作成してください。

## 前提
- NOTION_API_KEY 環境変数が設定済み
- 親ページはユーザーに確認して指定してもらう

## 作成するデータベースのプロパティ

| プロパティ名 | 型 | 設定 |
|-------------|------|------|
| 用語 | title | — |
| ステータス | select | 選択肢: 未処理, 処理中, 完了, 理解済み |
| カテゴリ | select | 選択肢: 技術, 一般, ビジネス |
| 処理日 | date | — |
| メモ | rich_text | — |

## 手順
1. Notion API で新規データベースを作成（POST /v1/databases）
2. 作成後、データベースの URL を表示
3. テスト用に「CRDT」「量子コンピュータ」の2件を追加
4. 「この URL を .env の NOTION_DB_URL に貼ってください」と案内
